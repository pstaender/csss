CoffeeScribtedStyleSheets

What can it do?

Compose stylesheets as usual - but without brackets and (semi)colons if you like

body
  background #eee
  a
    color grey
    font-weight bolder
    &:hover
      color black

Use variables - and you can append units if you like

@myColor = #eee
@myPaddingSize = 5px

@begin

body
  background @myColor
  padding @myPaddingSize * 2px

Use mixins

@transition = (@a) ->
  transition @a 0.2s
  -webkit-transition @a 0.2s
  -moz-transition @a 0.2s

@media all

a:hover
  @transition('color')
